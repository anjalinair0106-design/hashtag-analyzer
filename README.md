
# 📊 Hashtag Analyzer

A modern web app that analyzes hashtags and provides insights like popularity, related tags, and trends.

---

## 🚀 Features

- 🔥 **Popularity Score** — Instantly evaluates hashtag performance  
- 🔗 **Related Hashtags** — Suggests similar and trending tags  
- 📈 **Trend Visualization** — Displays weekly trend graph  
- ⚡ **Fast & Interactive UI** — Smooth and responsive experience  

---

## 🛠️ Tech Stack

- **Backend:** Flask (Python)
- **Frontend:** HTML, CSS, JavaScript
- **Visualization:** Chart.js
